# LocalSaveCard 개인정보 처리방침

**시행일:** 2024년 12월 19일

## 1. 개인정보 처리방침 개요

LocalSaveCard 앱(이하 "앱")은 사용자의 개인정보를 소중하게 생각하며, 최소한의 정보만을 수집하고 있습니다. 본 앱은 사용자의 개인정보를 수집하지 않으며, 모든 데이터는 사용자의 기기 내에서만 저장됩니다.

## 2. 수집하는 개인정보

**이 앱은 개인정보를 수집하지 않습니다.**

- 회원가입 불필요
- 계정 생성 불필요  
- 이메일, 전화번호, 이름 등 개인정보 수집 없음
- 사용자 식별 정보 수집 없음

## 3. 수집되는 정보 (개인정보가 아닌 정보)

### 앱 사용에 필요한 최소한의 정보만 수집:
- **기기 정보**: Android 버전, 기기 모델 (앱 호환성 확인용)
- **앱 사용 통계**: 앱 크래시 로그 (앱 안정성 개선용)
- **권한 사용**: 카메라 권한 (카드 스캔 기능용)

### 수집하지 않는 정보:
- 위치 정보
- 연락처 정보
- 통화 기록
- SMS 정보
- 브라우징 기록
- 외부 서버로의 데이터 전송

## 4. 데이터 저장 및 처리

### 로컬 데이터 저장:
- **카드 정보**: 사용자의 기기 내 메모리에만 저장 (앱 종료 시 삭제)
- **앱 설정**: 사용자의 기기 내에만 저장
- **OCR 처리**: 카드 스캔 시 기기 내에서만 처리

### 외부 서버 전송 없음:
- 모든 데이터는 사용자의 기기에서만 처리
- 클라우드 동기화 없음
- 외부 서버로 데이터 전송 없음
- 카드 정보는 외부로 전송되지 않음

## 5. 권한 사용 목적

### 카메라 권한 (CAMERA):
- **목적**: 카드 정보 스캔 및 OCR 인식을 위함
- **사용 범위**: 카드 스캔 기능 사용 시에만 사용
- **수집 정보**: 촬영된 이미지는 즉시 OCR 처리 후 삭제

## 6. 제3자 서비스

이 앱은 다음과 같은 제3자 서비스를 사용하지 않습니다:
- 분석 서비스 (Google Analytics 등)
- 광고 서비스
- 클라우드 저장 서비스
- 소셜 미디어 연동
- 외부 OCR 서비스 (모든 OCR 처리는 기기 내에서만 수행)

## 7. 데이터 보안

### 기술적 보안 조치:
- 모든 카드 정보는 기기 내에서만 처리
- 카드번호 마스킹으로 보안 강화
- 외부 네트워크를 통한 데이터 전송 없음
- 앱 종료 시 메모리에서 모든 카드 정보 자동 삭제
- 촬영된 카드 이미지는 OCR 처리 후 즉시 삭제

## 8. 사용자 권리

사용자는 다음 권리를 가집니다:
- **데이터 삭제**: 앱 종료 시 모든 카드 정보 자동 삭제
- **권한 관리**: 설정에서 카메라 권한 허용/거부 가능
- **데이터 접근**: 저장된 카드 정보는 앱 내에서만 확인 가능
- **카드 정보 편집**: 언제든지 카드 정보 수정/삭제 가능

## 9. 만 14세 미만 아동의 개인정보 보호

이 앱은 만 14세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다. 또한 개인정보를 수집하지 않는 앱이므로 아동 보호에 관한 특별한 조치가 필요하지 않습니다.
